# Next to Read: Concurrent and Follow-up Works

本論文（WhitenRec / WhitenRec+）は、逐次レコメンドにおいて異方性をもつテキスト埋め込みを「白色化」し、最適化と性能を改善する画期的なアプローチを示しました。
この論文を引用し、当該手法を発展・または別の角度からアプローチしている後続論文として、以下が挙げられます。

## 後続して読むべき関連論文リスト

1. **"Exploiting ID-Text Complementarity via Ensembling for Sequential Recommendation" (Collins et al., 2025)**
   - **理由**: 本論文が提案した「ID埋め込みは不要か？」という問いに対して、さらに踏み込み「IDとTextの両者は相互補完的に作用する」との立場からアプローチしているため。WhitenRecが取り組んだテキスト上の異方性問題を踏まえつつ、アンサンブルを用いてIDの持つ順序規則的な特長と協調させる研究として、ロジックの発展性を非常によく追うことができます。

2. **テキストベース逐次レコメンドにおける分布補正・コールドスタート推薦の最適化に関する最新研究**
   - **理由**: 本論文が扱った「コールドスタートにおける事前学習済み言語モデル推論（BERT-flowやZCAによる補正）」に影響を受け、大規模言語モデル（LLMs）で抽出した多次元モダリティ特徴をそのまま用いるためのアプローチ（Zero-Shot / Cold-Start）が多数派生しています。それらの分野のConcurrent workを追跡することで、IDフリーな推薦システムの最前線を体系的に理解する助けとなります。

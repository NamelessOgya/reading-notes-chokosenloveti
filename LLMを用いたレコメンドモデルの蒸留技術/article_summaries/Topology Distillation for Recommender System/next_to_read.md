# 次に読むべき論文 (Next to Read)

本研究「Topology Distillation for Recommender System (HTD)」を引用しており、さらにその手法を発展させたり、LLMという新しい切り口と関連付けた最新のアプローチを提案している後続の関連論文を以下に列挙します。

1. **Active Large Language Model-based Knowledge Distillation for Session-based Recommendation (ALKDRec)**
   - **概要**: 従来、トポロジーやエキスパートモデルから蒸留していた知識を、LLM（大規模言語モデル）の言語的推論能力から推薦モデルの系列表現へアクティブに蒸留させるというフレームワークです。Topology Distillationの「関係性を教示する」パラダイムを更に推し進め、LLMを通じた根拠や論理的関係性を小規模モデルに伝達しています。
   - **理由**: 「推薦モデルの知識蒸留」において、教師モデルを専門の推薦アルゴリズムからLLMへと進化させた最先端の事例であり、本研究（HTD）をベースライン・関連研究として強く引用して発展した系譜にあるため。

2. **Unbiased Knowledge Distillation for Recommendation**
   - **概要**: 知識蒸留を行う際、教師モデル自身が持っているバイアス（人気バイアス等）を生徒モデルへも引き継がせてしまう問題点を解決するための偏見を取り除く蒸留手法です。
   - **理由**: トポロジーのような高次関係性を写し取る技術が成熟した後に必ず直面する「教師モデルに潜むバイアスの伝播問題」へアプローチしており、関係性の蒸留をより実用に足る堅牢なモデルへ発展させる試みであるため。
